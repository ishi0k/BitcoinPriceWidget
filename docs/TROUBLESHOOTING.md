# Troubleshooting / Solución de problemas

## Español

### No llegan notificaciones
- Revisa permisos de notificaciones.
- Revisa optimización de batería (especialmente Xiaomi/Huawei/Samsung).
- Asegura que sonido/vibración estén habilitados si quieres esos efectos.

### El widget no se actualiza
- Verifica conexión.
- Aumenta el intervalo.
- Revisa permisos de “alarma exacta” si tu Android lo solicita.

### Me sale “expired / trial expired”
- Tu TRIAL venció o tu PRO requiere renovación.
- Activa un nuevo código o renueva.

## English
- Notifications: check permissions + battery optimization.
- Updates: connectivity + interval + exact alarm permission.
