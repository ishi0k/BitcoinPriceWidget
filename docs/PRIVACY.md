# Privacy / Privacidad

## Español
Esta app no solicita email ni crea cuentas tradicionales.

Para operar el widget y el licenciamiento, el backend puede recibir:
- Un identificador de dispositivo (device id) usado para *binding* de licencia.
- `app_id` (nombre del paquete) y firma del APK (apk signature) para validar que la app no fue reempaquetada.
- Metadatos básicos del dispositivo (modelo/fabricante/versión de Android) para diagnóstico.

No se recopilan:
- Contactos
- Ubicación GPS
- Fotos/archivos personales

## English
No email or traditional accounts are required.

For widget operation and licensing, the backend may receive:
- A device identifier used for license binding.
- `app_id` and APK signature to reduce re-packaging abuse.
- Basic device metadata (model/manufacturer/Android version) for diagnostics.

Not collected:
- Contacts
- GPS location
- Personal photos/files
